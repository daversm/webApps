Permutation App


Permutation can be summed up by asking the question, how many
different ways can we arrange a set, when order does matter?
So if we have the set {1,2}, then the arrangments are {1,2} and {2,1}
The formula is simple : n!/ (n-k)! , n being the number of elemnts in
a set, and k is the length of the arrangments.

Here we have a set of three unique cards, how many permutations

problem is to imagin 3 slots or placeholders. Three placeholders,
since in our set of cards we have three elements. The reason why 3!
gives us the total number of permutations is beacuse for the first
placeholder we have three cards we can pick from , for the second
placehold we only have 2 cards to pick from and for the last
placeholder there is just one card left.(3 x 2 x 1 = 3! = 6) hopfully that
makes sense , if not give it another read

Combination App


Much like permutation, combination also involves aranging things.
However the difference is that unlike permutation, order does not matter.
The formula for combination is: n! / (r!(n-r)!). n is the number of elements
in a set, and r is the length, or the the number of elements used in an
arangment.
