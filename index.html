Permutation App


Permutation can be summed up by asking the question, how many
different ways can we arrange a set, when order does matter?
So if we have the set {1,2}, then the arrangments are {1,2} and {2,1}
The formula is simple : N! , N being the number of elemnts in a set.

Here we have three unique cards, how many permutations are there?
Using the formula for total number of permutations we get 6 (3! = 6).
A good way to approce this problem is to imagin 3 slots, or placeholders.
Three placeholders, since in our set of cards we have three elements
The reason why 3! gives us the total number of permutations is beacuse
for the first place holder we have three cards we can pick from , for the
second placehold we only have 2 cards to pick from and for the last
placeholder there is just one card left.(3 x 2 x 1 = 3! = 6) hopfully that
